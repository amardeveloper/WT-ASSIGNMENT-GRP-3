HTML FORMATTING ELEMENTS

HTML uses elements like < b > and < i > for formatting output, like bold or italic text.

Formatting elements were designed to display special types of text:
  • < b > - Bold text
  • < strong > - Important text
  • < i > - Italic text
  • < em > - Emphasized text
  • < mark > - Marked text
  • < small > - Small text
  • < del > - Deleted text
  • < ins > - Inserted text
  • < sub > - Subscript text
  • < sup > - Superscript text

HTML < b > and < strong > Elements

The HTML < b > element defines bold text, without any extra importance.

Example

< b >This text is bold< /b >

Output

This text is bold

The HTML < strong > element defines strong text, with added semantic "strong" importance.

Example

< strong >This text is bold< /strong >

Output

This text is strong

HTML < i > and < em > Elements

The HTML < i > element defines italic text, without any extra importance.

Example

< i >This text is bold< /i >

Output

This text is italic

 The HTML < em > element defines emphasized text, with added semantic importance.

Example

< em >This text is emphasized< /em >

Output

This text is emphasized

Note: Browsers display < strong > as < b >, and < em > as < i >. However, there is a difference in the meaning of these tags: < b > and < i > defines bold and italic text, but < strong > and < em > means that the text is "important".

HTML < small > Element

HTML < small >small< /small >formatting

HTML < mark > Element

The HTML < mark > element defines marked or highlighted text:

HTML < del > Element

The HTML < del > element defines deleted (removed) text.

Example

< p >My favorite color is < del >blue< /del > red.< /p >

Output

My favorite color is blue red.

HTML < ins > Element

The HTML < ins > element defines inserted (added) text.

Example

< p >My favorite color is < ins >blue< /ins > red.< /p >

Output

My favorite color is blue red.

HTML < sub > Element

The HTML < sub > element defines subscripted text.

Example

< p >This is < sub >subscripted< /sub > text.< /p >

Output

This is subscripted text.

HTML < sup > Element

The HTML < sup > element defines superscripted text.

Example

< p >This is < sup >subscripted< /sup > text.< /p >

Output

This is subscripted text.
